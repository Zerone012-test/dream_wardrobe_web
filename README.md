# 梦境衣橱 · 情绪换装网站

情绪像衣服一样穿在身上。有些衣服越穿越沉——但衣服，是可以换的。

Y3K 风格服装 × 情绪困境具象化，内置 DeepSeek AI 生成个性化文案。

---

## 快速启动

```bash
npm install
cp .env.example .env     # 填入你的 DEEPSEEK_API_KEY
npm start                 # 访问 http://localhost:3000
```

> DeepSeek API Key：https://platform.deepseek.com

## 如何修改 Agent Prompt

所有 AI 生成文案的 prompt 都在 `public/index.html` JS 部分，搜索以下注释定位：

| 注释标记 | 对应功能 | 字数 |
|---------|---------|------|
| `/* ── AGENT PROMPT: 换装解读 */` | 抽卡页：脱下→换上的意义解读 | 约150字 |
| `/* ── AGENT PROMPT: 换上之后的信 */` | 信笺页：个人化换装信 | 约200字 |

结果页的服装×困境解读 prompt 在 `showResult()` 函数内 `callAgent('agent-result', ...)` 调用处。

Agent 的**人格基调和世界观**（情绪像衣服可以换）在 `server/index.js` 的 `SYSTEM_PROMPT` 常量中修改。

## 三板块说明

| 板块 | 导航名 | 功能 |
|------|--------|------|
| **01 看见** | 看见 | 弹窗填写姓名 → 15道测试题 → 展示困境类型+口语人格+困境服装全宽展示+AI解读服装×情绪联结 |
| **02 换一件** | 换一件 | 15张牌扇形展开（命运牌有光点）→ 选牌右侧同步预览（不重叠）→ 确认后其余牌消散 → 翻转揭示换装服装+AI解读换上的意义 |
| **03 换上之后** | 换上之后 | 右侧两张服装卡片交叠展示（脱下的/换上的）+ 打字机效果个人化信笺 |

## 故事逻辑

```
情绪像衣服一样穿在身上
    ↓
测试 → 找出你现在穿着哪种情绪（困境服装）
    ↓
困境服装 → 看见那件穿在身上的衣服长什么样（Y3K具象化）
    ↓
抽卡 → 凭感觉选一件想换上的（命运早已决定，但需要你来确认）
    ↓
翻转揭示 → 换上它（服装设计如何体现这种转变）
    ↓
信笺 → 换衣服之后，有人写信告诉你这件衣服适合你
```
